Entrega proyecto final - Curso front End
Aquí se encuentra el proyecto final de “Primeros Pasos en el Desarrollo Front End” para Argentina Programa 4.0, de la escuela Ticmas.
https://cershuli.github.io/

# Sobre el proyecto
- Se realizó la vinculación de HTML, CSS y JavaScript.
- Se logró adicionar imágenes.
- Formato full responsive.
- Animaciones en botones y al pasar el Mouse sobre elementos en específico (En zona de educación).
- Los elementos se acondicionan a tamaño de escritorio y dispositivos móviles (Por ejemplo, "La barra de navegación" y zona de "contacto").
- Los botones redireccionan a sus respectivas páginas.
- Diseño e inserción de Logotipo propio.
- Aunque no cuento con formación en el área de diseño, intente utilizar una selección de colores tanto representativa como adecuada.
- Se logró incluir una barra de navegación, banner con contenido, además de un pie de página.
- La zona de Habilidades e idiomas cuenta con animacion.

# Mi Experiencia
Este fue mi primer acercamiento a un proyecto de estas características, lo cual presentó un verdadero reto para mi persona. Me enfoque bastante en que el proyecto sea amigable a la vista además de utilizar la mayor cantidad de elementos que vimos en el curso, intentando que sea similar a las páginas a las cuales estamos acostumbrado a utilizar comúnmente, con animaciones y acondicionamiento a varios tipos de formato. 

El proceso de creación fue largo y entretenido con una gran carga horaria, disfruté mucho el proceso de diseño y utilización de CSS y cómo cada elemento variaba en función de cada característica. El pasar del archivo HTML, al agregar el CSS y realmente parece mágico, el ver como varía con cada cambio, el descubrir nuevas funcionalidades y ver cada sección como uno lo imagina realmente da una gran gratificación. Añadí títulos a cada sección para una mayor facilidad a la hora de encontrar o acomodar código.

Estoy bastante satisfecho con el curso, si es que cumplo con los requisitos de este, me voy a seguir informando y capacitando en front end, y profundizaré en JS y similares. La página cuenta con mi correo de gmail, el cual la persona que vea mi proyecto y quiera comentarme su opinión o tenga algún tipo de consulta, lo invito a hacerlo.

Aunque es una página de grado inicial, estoy orgulloso y satisfecho con mi proyecto, esto es solo el comienzo, estaré agradecido de cualquier tipo de recomendación y/o consejo.
